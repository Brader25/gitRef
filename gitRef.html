Brads Git Cheat Sheet

Action Command
Change current file cd (file you would like to use)
Change directory to your root file cd /
Move up a level from your current directory cd ..
Show all files in the current directory ls
View files that are hidden in current directory ls -a
Initialize a stand-alone, local Git repository git init
Initialize a local Git repository from a GitHub repository git clone "Url from repository"
Get the most recent code from GitHub git pull "Url to repository"
Stage changed files that you want to be commited git add .
Commit the changes to your file. git commit -m "(your commit message here)"
Check the status of the repository git status
Save your changes to a repository out on GitHub git push
View a list of all the commits in a repository git log
Make a temporary branch from the master repository git checkout -b (branch name)
Merge branch into master git checkout master; git merge (branch nmae)
Link to gitRef repository.
